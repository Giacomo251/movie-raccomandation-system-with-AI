# Movie raccomandation system with AI
A simple recommendation system that, given a movie as input, can recommend similar films.

The CSV file consists of the following columns: ID, plot, year, genre, language, rating, country, budget, and revenue. To recommend, it uses the plot (60%), genre (30%), and year, rating, budget, and revenue (10%).
Csv file: https://www.kaggle.com/datasets/ashpalsingh1525/imdb-movies-dataset
